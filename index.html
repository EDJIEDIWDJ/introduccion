Competencias

01

Scan Station

Scan Station,es un software de entrada y salida para los empleados. La instalación de este software de usar, compatible con Window 7,8,9. Los productos Scan Station principalmente están ubicados en Estados Unidos, existen varias páginas web donde se puede descargar directamente el software.

02

Aguila Doce

Es un software especializado para el control de la entrada y salida de los visitantes en una determinada empresa. Águila Doce permite la integración de un registro biométrico y fotográfico a la información básica de los visitantes, mediante dispositivos Biométricos y otros tipos de identificaciones

03

Softcontrol

Otro competidor es Softcontrol que es una compañía que lleva 13 años de experiencia en la implementación y desarrollo de software empresariales para identificación, seguridad y control de tiempos del personal, utilizando las últimas tecnologías biométricas existentes a nivel mundial.
